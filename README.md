### System 

-  Linux
-  Windows ( CMD ) 
-  Android ( Termux ) 

### Install Tool

```
-  pkg update && pkg upgrade
-  pkg install python -y
-  pkg install python2 -y
-  pkg install git -y
-  git clone https://github.com/kanekikon/checker
```


### Run

```
[-] cd checker 
[-] chmod +x checker.py
[-] python checker.py
```
